Python Style Guide
=========================

## 이름규칙
- 모든 변수와 함수 이름은 소문자로, 여러 단어일 경우 _로 나누기.
- 모든 상수 이름은 대문자로, 여러단어일 경우 _로 나누기.
- 의미있는 변수/함수 이름 사용.

## 화이트스페이스
- 적당한 화이트 스페이스는 코드를 일기 편하게 한다.
  - ex) 연산자 앞뒤  print(2 * 3 * 4)
- 글 쓸때 문단을 나누듯이 프로그램을 짤때도 엔터키를 써서 적당히 나누기.

## 적당한 추상화
- 변수/함수 등을 적절하게 사용.
